# My-Parking-Assistant
Web app which help to find out our car where we parked last in a congested parking area
This is a simple web application that helps users remember where they parked their vehicle.

## Features
- Save parking location using browser geolocation
- Retrieve location later
- Open location directly in Google Maps

## Tech Stack
- Python (Flask)
- HTML/CSS
- Browser Geolocation API
- Google Maps

## How it works
1. User clicks "Save Parking Location"
2. Location is stored temporarily
3. User asks "Where did I park?"
4. App returns a Google Maps link
